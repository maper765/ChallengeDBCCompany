# ChallengeDBCCompany
Teste para Desenvolvedor

>Aplicação console desenvolvida em .NET Core 3.1. 

#### Passos para Execução
1. Modifique `_pathIn` e `_pathOut` de acordo com o caminho correspondente
2. De start na aplicação e coloque os arquivos no diretório `_pathIn`
3. Verifique as saídas em `_pathOut`
4. Adicione mais arquivos no diretóriod e entrada, enquanto a aplicação console estiver em execução, os arquivos serão processados
