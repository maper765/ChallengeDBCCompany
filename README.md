# ChallengeDBCCompany
### Teste para Desenvolvedor

Criar um sistema de análise de dados, onde deverá ser importados lotes de arquivos, ler e analisar os dados e produzir um relatório.

>Aplicação console desenvolvida em .NET Core 3.1 com C#. 

#### Passos para Execução
1. Em `Program.cs`, modifique `_pathIn` e `_pathOut` de acordo com o caminho correspondente em sua máquina
2. De start na aplicação 
3. Coloque os arquivos `.dat` no diretório `_pathIn`
4. Verifique as saídas de relatórios no diretório `_pathOut`
5. Adicione mais arquivos ao diretório de entrada, enquanto a aplicação console estiver em execução, os arquivos serão processados
