# ChallengeDBCCompany
Teste para Desenvolvedor

>Aplicação console desenvolvida em .NET Core 3.1. 

#### Passos para Execução
1. Modifique `_pathIn` e `_pathOut` de acordo com o caminho correspondente em sua máquina
2. De start na aplicação e coloque os arquivos no diretório `_pathIn`
3. Verifique as saídas no diretório `_pathOut`
4. Adicione mais arquivos ao diretório de entrada, enquanto a aplicação console estiver em execução, os arquivos serão processados
